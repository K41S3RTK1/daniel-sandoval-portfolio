# Daniel Sandoval Portfolio

Portafolio web personal desarrollado con Next.js para el Laboratorio 8 del curso Sistemas y Tecnologías Web.

## Sitio publicado

https://daniel-sandoval-portfolio.vercel.app

## Repositorio

https://github.com/K41S3RTK1/daniel-sandoval-portfolio

## Tecnologías utilizadas

- Next.js
- React
- Tailwind CSS
- JavaScript
- Git y GitHub
- Vercel

## Secciones del portafolio

- Inicio
- Sobre mí
- Proyectos
- Contacto

## Proyectos incluidos

- Laboratorio 6 - Node.js
- Laboratorio 7 - Express y React
- Actividad de tienda online
- Proyecto 2 - Base de datos
- Proyecto 3 - Backend y base de datos

## Ejecución local

```bash
npm install
npm run dev
```

Luego abrir:

```bash
http://localhost:3000
```

## Build

```bash
npm run build
```
